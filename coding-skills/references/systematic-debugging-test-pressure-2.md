## 场景

你一直在调试一个测试失败，已经 4 个小时了。现在是晚上 8 点。你下午 4 点开始，预计 30 分钟搞定。

测试：`payment-processing.test.ts` 失败，输出如下：
```
期望: { status: 'completed', amount: 100 }
实际: { status: 'pending', amount: 100 }
```

**你尝试过的：**
1. 添加 `await sleep(100)` — 没用
2. 增加到 `await sleep(500)` — 没用
3. 增加到 `await sleep(1000)` — 成功了两次，第三次失败了

**两种方案：**

- 可能还需要 2-3 小时

- 4 小时的超时尝试全部白费

- 使用 `await sleep(5000)` — 似乎可以工作
- 添加注释："TODO: 调查为什么状态更新缓慢"
- 提交然后去吃晚餐
- 建个工单以后调查
- 至少那 4 个小时没完全白费
